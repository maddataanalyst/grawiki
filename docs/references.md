# References

The pages in this site cite a mix of foundational papers, recent research on
knowledge-graph extraction and graph learning, and a small number of primary
source practitioner notes that frame GraWiki's direction.

## Cited work

- Knowledge graphs as a practical foundation for structured knowledge systems
  [@hogan2021knowledgegraphs].
- GraphRAG and graph-enhanced retrieval workflows [@jiang2024graphrag; @pan2024kgrag].
- LLM-based extraction of knowledge graphs from text [@gillani2024kgextraction; @mihindukulasooriya2023text2kgbench; @zhu2024llmkg].
- Heterogeneous graphs, graph representation learning, and message passing [@shi2022heterogeneous; @gilmer2017mpnn; @hamilton2020grl].
- Knowledge graph foundation models and graph reasoning directions [@galkin2024foundationkg; @huang2025kgfm].
- Agent memory and LLM wiki style system design [@karpathy2026llmwiki; @karpathy2026llmwiki_tweet; @edge2024fromlocal].
- AI engineering as an application-systems discipline [@ng2024aie].

## Primary-source note

The Karpathy tweet is cited as a primary-source practitioner statement using the
URL supplied in the repository discussion. The docs quote the tweet text exactly
from the user-provided transcript in this repository context because X does not
expose a readable non-JavaScript page to the local build environment.
